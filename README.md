  ## Technologies & Tools

  - **GeoPandas** — spatial data manipulation and analysis
  - **Folium / Plotly** — interactive geospatial visualisations and mapping
  - **Python** — pandas, NumPy, Matplotlib, SciPy
  - **Jupyter Notebooks** — interactive spatial analysis and reporting
  - **QGIS** — geographic information system for spatial data exploration

  ## Module Overview

  Geospatial Analysis bridges data science and geographic information science (GIS). Key topics include:

  - Coordinate reference systems and spatial data formats (GeoJSON, Shapefile)
  - Spatial joins, overlays, and proximity analysis
  - Choropleth mapping and spatial visualisation
  - Point pattern analysis and spatial clustering
  - Real-world applications: urban planning, environmental monitoring, humanitarian data
